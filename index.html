create-project

🛠️ 鲸腾FE工程开发的标准工具

快速上手 项目简介

大道至简

以 Vue 为基础框架，纯净的空白模板，塑造更便捷的前端开发环境。

功能丰富

一键创建 Web、小程序、H5、离线包应用程序，快捷开发，开箱即用。

生态丰富

可以选择 JavaScript 作为开发语言，也支持 TypeScript，同时提供常用的 UI 组件库支持。

构建打包工具

默认的打包工具是 Vite ，也同样可以选择 Webpack。

开箱即用

内置了路由、请求、移动端适配等，仅需安装依赖即可上手开发。内含丰富的功能，可满足日常 80% 的开发需求。

面向未来

在满足需求的同时，我们也不会停止对新技术的探索。比如跨端、微前端、webpack@5、bundle less 等等。

# 像数 1, 2, 3 一样容易

# 在你的项目中安装
npm init @winner-fed/project@latest my-project

# 进入工程
cd my-project

# 安装依赖
npm run bootstrap

# 开发调试
npm run serve

# 构建打包
npm run build
